	Lately I was fascinated with all the success of [AvatarAI](https://avatarai.me/) by [@levesio](https://twitter.com/levelsio) and wondered what powered his service. And thus I entered the rabbit hole of Stable Diffusion.

	## Install Docker on Debian 11
	```sh
	$ sudo apt-get install ca-certificates curl gnupg lsb-release
	$ sudo mkdir -p /etc/apt/keyrings
	$ curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
	$ echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/debian bullseye stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
	$ sudo apt-get update
	$ sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
	$ sudo docker run hello-world
	```

	## Install NVIDIA Container Toolkit
	```sh
	$ curl -fsSL https://nvidia.github.io/libnvidia-container/gpgkey | sudo gpg --dearmor -o /usr/share/keyrings/nvidia-container-toolkit-keyring.gpg \
	  && curl -s -L https://nvidia.github.io/libnvidia-container/debian11/libnvidia-container.list | \
	  sed 's#deb https://#deb [signed-by=/usr/share/keyrings/nvidia-container-toolkit-keyring.gpg] https://#g' | \
	  sudo tee /etc/apt/sources.list.d/nvidia-container-toolkit.list
	$ sudo apt-get update
	$ sudo apt-get install -y nvidia-docker2
	$ sudo systemctl restart docker
	$ sudo docker run --rm --gpus all nvidia/cuda:11.6.2-base-ubuntu20.04 nvidia-smi
	```

	## Clone Stable Diffusion repository
	```sh
	$ git clone https://github.com/basujindal/stable-diffusion ~/stable-diffusion
	$ mkdir -p ~/{sd-data,sd-output}
	$ cd ~/stable-diffusion
	$ docker compose up --detach
	```
